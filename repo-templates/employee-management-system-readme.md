## 🚀 Features
- **CRUD Operations**: Add, read, update, and delete employee records securely.
- **Database Integration**: Seamless connection with MySQL using JDBC.
- **Responsive UI**: Built with Java Swing/JavaFX for a clean, user-friendly interface.

## 🛠️ Setup Instructions
1. Clone this repository.
2. Execute the provided `schema.sql` in your MySQL instance to create the database.
3. Update `db.properties` with your database credentials.
4. Run the application via your IDE or Maven `mvn exec:java`.
