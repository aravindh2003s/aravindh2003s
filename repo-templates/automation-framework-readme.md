## 🌟 Architecture
- **Page Object Model (POM)**: Enhanced maintainability by separating logic and UI locators.
- **Data-Driven**: Inputs are supplied securely via Excel/JSON integrations.
- **Cross-Browser**: Supports Chrome, Edge, and Firefox seamlessly.
- **Detailed Reporting**: Integrated with ExtentReports for rich HTML test results.

## 🚀 Execution
Run the entire suite using Maven:
```bash
mvn clean test -DsuiteXmlFile=testng.xml
```
